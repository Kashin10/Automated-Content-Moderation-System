
# Automated Content Moderation System

## Overview

Cloud-native automated content moderation system built using AWS, Docker, Jenkins, OCR pipelines, and real-time monitoring.

## Features

- OCR-based image and text moderation
- CI/CD automation using Jenkins
- Dockerized deployment
- AWS EC2 hosting
- S3 storage integration
- CloudWatch monitoring
- SNS notifications
- Real-time moderation workflow

## Architecture

GitHub → Jenkins → Docker → EC2 Deployment

## Tech Stack

### Cloud
